# AI Lab Assignments

A structured repository for my Fontys ICT AI lab assignments.

## Goals

- Keep all lab assignments organized in one place
- Work mainly with JupyterLab
- Make learning more fun and memorable

## Structure

- `notebooks/` → Jupyter notebooks for each lab
- `src/` → reusable helper code
- `data/` → datasets used in assignments
- `assets/` → images, diagrams, funny extras
- `notes/` → explanations, question bank, ideas

## Environment

This project uses a Python virtual environment and JupyterLab.

## How to run

```bash
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
jupyter lab
